Centipede Project

This is a project that I have been working on for a while now.

It is a simple scraping full stack application that will allow you execute your own python and javascript languages using an API.

The application also have a docker for a mysql database and any technology that you might want to test will probably work.

The API executes the spawn command making children nodes and executes those scripts based on your stack, you can execute frameworks also like puppeteer or selenium.

The application is still in development and I am working on the front end to make it more user friendly.

The application is open source and you can find it on my github.
